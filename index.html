Cheat Sheet HTML

<!DOCTYPE html>

Define el tipo de documento y versión de HTML.

<!DOCTYPE html>
<html>

Elemento raíz de un documento HTML.

<html lang="es"> ... </html>
<head>

Contiene metadatos, enlaces a CSS, scripts y el título.

<head><title>Mi Página</title></head>
<body>

Contiene el contenido visible de la página.

<body> ... </body>
<title>

Define el título de la pestaña del navegador.

<title>Mi Página</title>
<header>

Sección de encabezado de la página o sección.

<header><h1>Bienvenido</h1></header>
<nav>

Contiene enlaces de navegación.

<nav> <a href="#">Inicio</a> <a href="#">Contacto</a> </nav>
<main>

Contenido principal de la página.

<main><p>Contenido principal</p></main>
<aside>

Contenido lateral o complementario.

<aside><p>Publicidad o enlaces</p></aside>
<footer>

Pie de página de la web.

<footer><p>© 2025</p></footer>
<h1> - <h6>

Encabezados de diferentes niveles.

<h1>Título 1</h1> <h2>Título 2</h2>
<span>

Elemento en línea para aplicar estilos.

<p>Texto con <span style="color:red">resaltado</span></p>
<div>

Contenedor de bloque, útil para agrupar elementos.

<div class="caja">Contenido</div>
<a>

Enlace a otra página o recurso.

<a href="https://google.com">Ir a Google</a>
<ol>, <ul>, <dl>

Listas: ordenadas, desordenadas y de definición.

<ul><li>Item</li></ul> <ol><li>Item</li></ol> <dl><dt>Término</dt><dd>Definición</dd></dl>
<table>, <tr>, <td>, <th>, <thead>, <tbody>, <tfoot>

Tablas y sus secciones.

<table> <thead><tr><th>Encabezado</th></tr></thead> <tbody><tr><td>Dato</td></tr></tbody> <tfoot><tr><td>Pie</td></tr></tfoot> </table>
<img>, <audio>, <video>, <source>

Multimedia: imágenes, audio y video.

<img src="img.jpg" alt="Imagen"> <audio controls><source src="audio.mp3" type="audio/mp3"></audio> <video controls width="300"><source src="video.mp4" type="video/mp4"></video>
<form>, <input>, <select>, <option>, <textarea>, <label>

Formulario y campos de entrada.

<form action="#" method="post"> <label>Nombre:</label> <input type="text" name="nombre"> <label>Sexo:</label> <select> <option>Masculino</option> <option>Femenino</option> </select> <label>Mensaje:</label> <textarea></textarea> <input type="submit" value="Enviar"> </form>
